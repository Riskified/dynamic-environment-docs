# Supported Versions

_Dynamic Environment_ is currently tested against the following matrix of kubernetes / Istio
versions:

| Kubernetes Version | Istio Version |
|--------------------|---------------|
| `v1.21.x`          | `v1.12.2`     |
| `v1.23.x`          | `v1.14.6`     |
| `v1.25.x`          | `v1.16.2`     |

Any combination of these versions should work.
